# ☕ Coffee Sales Data Analysis using SQL

A complete **SQL-based Data Analysis Project** that explores coffee sales data to generate meaningful business insights. This project demonstrates strong skills in **data analysis, SQL querying, and real-world problem solving**.

The analysis focuses on **customer behavior, product performance, revenue trends, and market opportunities** across multiple cities.

---

## 🚀 Project Overview

This project analyzes coffee sales data using SQL to answer real-world business questions such as:

- Revenue generation trends  
- Customer segmentation  
- Product performance analysis  
- Market potential across cities  
- Sales growth over time  

The project simulates how data analysts extract insights to support **business decision-making**.

---

## 🎯 Objectives

- Perform **Exploratory Data Analysis (EDA)** using SQL  
- Analyze **sales, customers, and product data**  
- Generate **business insights using advanced queries**  
- Apply **window functions and aggregations**  
- Identify **growth trends and market opportunities**  

---

## 🗂️ Database Schema

The project uses four main tables:

- **City** – City details, population, rent, ranking  
- **Customers** – Customer information and city mapping  
- **Products** – Coffee product details and pricing  
- **Sales** – Transaction data including date, revenue, and ratings  

📌 Proper **foreign key relationships** ensure data integrity and realistic modeling.
